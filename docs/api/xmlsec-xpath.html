xmlSecXPathHereFunction ()

void        xmlSecXPathHereFunction         (xmlXPathParserContextPtr ctxt,
                                             int nargs);

The implementation of XPath "here()" function. See xmlXPtrHereFunction() in xpointer.c. the only change is that we return NodeSet instead of NodeInterval.

ctxt : the pointer to XPath parser context.
nargs : the args number.
