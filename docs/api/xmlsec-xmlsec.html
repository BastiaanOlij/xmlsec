Description

Details

xmlSecInit ()

void        xmlSecInit                      (void);

Initializes XML Security Library. The depended libraries (LibXML, LibXSLT and OpenSSL) must be initialized before.

xmlSecShutdown ()

void        xmlSecShutdown                  (void);

Clean ups the XML Security Library.
