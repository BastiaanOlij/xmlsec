    const xmlChar			*href;

    xmlSecTransformDestroyMethod	destroy;

xmlSecDigestVerifyNode ()

int         xmlSecDigestVerifyNode          (xmlSecTransformPtr transform,
                                             const xmlNodePtr valueNode);

Reads the node content, base64 decodes it, finalizes the digest result and verifies that it does match with the content of the node. This functions returns only operation status (ok, fail). Verification status is stored in xmlSecTransform::state variable
