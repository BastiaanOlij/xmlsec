Download

The latest XML Security Library version is 0.0.13:
  • Sources (XMLSoft.org and RPMFind.net mirrors).
  • Red Hat RPM xmlsec and xmlsec-devel packages (compiled with OpenSSL 0.9.6 because there are no OpenSSL 0.9.7 RPM yet; have incomplete XML Encryption functionality).
  • Windows binaries for XMLSec library (as well as LibXML2, LibXSLT and OpenSSL) from Igor Zlatkovic.
  • Un-official XMLSec Debian packages from John Belmonte.
The daily snapshots and previous versions are also available.

ATTENTION
I am doing some major code refactoring these days. The snapshot builds most likely are broken. Please, use 0.0.13 version instead.
If you want to use sources from GNOME CVS do not check out the tip. It is broken and most likely does not compile. The stable 0.0.13+ branch is XMLSEC_0_0_X_BRANCH.

Requirements

The XML Security Library requires

  • LibXML version 2.4.24 or geater
  • LibXSLT 1.0.20 or greater (optional)
  • OpenSSL version 0.9.7 (prefered) or OpenSSL version 0.9.6c

Installation (source archive)

All steps are usual:

gunzip -c xmlsec-xxx.tar.gz | tar xvf -
cd xmlsec-xxxx
./configure --help
./configure [possible options]
make
make install
make check
The last step is optional and requires Internet connection to execute some tests.

CVS

XML Security Library is also available from the Gnome CVS (the module name is xmlsec). The online source browser: Gnome CVS base .
